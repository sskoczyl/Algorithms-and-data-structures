## Algorithms and data structures  
During this course at university following algorithms/structures were implemented:
* Sorting alghorithms:
  * Insertion sort
  * Quick sort
  * Merge sort
  * Tim sort
  * Count sort
  * Radix sort
  * Selection sort
* Search algoritms:
  * Binary search
  * Quickselect
* Graph alghorithms:
  * Dijkstra
  * Kruskal
  * DFS and BFS
* Structures:
  * Queues
  * Linked lists
  * Graphs
  * Binary search trees
  * Red-black trees
  * Hash maps
  * Bloom filter
